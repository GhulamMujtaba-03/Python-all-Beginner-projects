# Python Beginner Projects

This repository contains All the beginner-friendly Python projects. Each project is fully functional and comes with instructions to run.

---

##  Simple Calculator-Project1

### Description
A simple calculator program that allows the user to perform basic arithmetic operations: addition, subtraction, multiplication, and division.

### Features
- Addition, subtraction, multiplication, division
- Input validation to ensure numbers are entered correctly
- Continuous calculation until the user decides to exit

### How to Run
1. Open the file `calculator.py` in Python.
2. Run the program.
3. Follow on-screen instructions to perform calculations.

##  Number Guessing Game-Project3

### Description
A fun number guessing game where the computer selects a random number between 1 and 100. The user has to guess the number, receiving feedback if their guess is too high or too low.

### Features
- Random number selection between 1 and 100
- Tracks the number of attempts
- Input validation for numbers only
- Option to play again after completing the game

### How to Run
1. Open the file `guess_game.py` in Python.
2. Run the file.
3. Follow prompts to guess the number.
4. After guessing correctly, you can choose to play again.



# Rock-Paper-Scissors Game-project4

### Description
A fun command-line game where you play **Rock, Paper, Scissors** against the computer. The computer randomly selects its choice, and you try to beat it by choosing rock, paper, or scissors.

### Features
- Play against the computer
- Random computer choice
- Shows the winner or a tie
- Option to quit anytime

### How to Run
1. Open the file `rock_paper_scissors.py` in Python.
2. Run the file.
3. Enter your choice: rock, paper, or scissors.
4. After each round, see the result and play again or type 'quit' to exit.

# Dice Rolling Game-Project5
### Description
A simple simulator that rolls a dice and displays a random number between 1 and 6. Press Enter to roll and enjoy a fun random outcome.

### Features
- Rolls a dice with a number between 1 and 6
- Press Enter to roll again
- Option to quit anytime
- Simple and interactive for beginners

### How to Run
1. Open the file `dice_simulator.py` in Python.
2. Run the file.
3. Press Enter to roll the dice.
4. Type 'quit' anytime to exit the program.
